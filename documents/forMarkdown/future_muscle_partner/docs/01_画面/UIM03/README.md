# [UIM03] カレンダー予約

## 概要

機能目的:

- トレーニーやパーソナルトレーナの空き時間を表示し、予約を行えるようにする

機能概要:

- 本日以降の週間カレンダーを表示する
- すでに予約済みの時間がわかるようにする
