# [UIM04] 決済

## 概要

機能目的:

- トレーニング予約内容を表示し、決済によって確定する

機能概要:

- トレーニングのコース／日時／料金を表示
- クレジット決済に必要な項目の入力
- 決済の確定

## イベント概要

| No | イベント名   | イベント分類 | 処理説明      |
|----|---------|--------|-----------|
| 1  | 初期表示    | 初期表示   |           |
| 2  | 日時指定に戻る | ボタン押下  | UIM003に遷移 |
| 2  | 決済する    | ボタン押下  |           |
